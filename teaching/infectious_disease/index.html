If you hover over the top right corner of a code block in these lecture notes, a copy button emerges that allows you to copy the code (and then paste it into Rstudio) without having to first select all of the code.

1.2 What to work on?

Look at the sections that are listed as Task. Please try to solve those tasks and put resulting code and figures in a word doc or use an rmarkdown document to compile your answers.
